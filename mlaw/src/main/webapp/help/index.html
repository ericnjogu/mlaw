Event Lookup

The Location and Comment fields can accept exact values e.g. milimani or meeting.
Wildcards can also be used for these fields to match data containing the specified value. e.g.
*court* - match any data containing the word court.
*meeting* - match any data containing the word meeting.

Start Date and End Date fields work best with a date range as below. Use the date button to select the first date then edit by hand.
02 Mar 2013 .. 01 Apr 2013 searches for all dates between the two dates (inclusive)
> 02 Mar 2013 searches for dates newer (more recent) than the date given
< 02 Mar 2013 searches for dates older (less recent) than the date given

Notes and Attachments

Please add any files that have more information relating to this action e.g. file(s) that represent the work you have done. Strictly Confidential information should be encrypted before being uploaded.

Files that were existing in soft or hard copy before this matter was imported into mLaw can be added here.
Correspondence that has been stamped as 'RECEIVED' may also be scanned and added.

You can drag and drop a file or internet link to the file upload or note text.
; Copying and pasting a link from an email, document, file explorer, browser or any source should work.
